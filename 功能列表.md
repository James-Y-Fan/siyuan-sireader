# 📖 SiReader 插件功能总结表

| **📚 阅读** | 格式支持 | EPUB/PDF/TXT/在线小说 |
| | 主题配色 | 8种预设主题（默认/杏仁黄/秋叶褐/青草绿/海天蓝/夜间/暗黑/赤金）+ 自定义主题 |
| | 阅读模式 | 单页/双页/连续滚动 |
| | 翻页动航 | 滑动/滚动/无动画 |
| | 打开方式 | 新标签/右侧标签/底部标签/新窗口 |
| | 导航栏位置 | 左侧/右侧/顶部/底部，支持自定义功能模块和排序 |
| | 目录导航 | 目录/书签/标记三种模式，支持搜索章节、反序、跳转 |
| | 脚注识别 | 自动识别脚注/尾注/参考文献/术语，点击弹窗显示 |
| | 文本设置 | 字体/字号/字距/行距/段距/首行缩进 |
| | 布局设置 | 左右边距/上下边距/列间距/页眉页脚高度/最大内容宽度 |
| | 视觉增强 | 亮度/对比度/复古色调/饱和度/反色模式 |
| | 阅读统计 | 本次/今日/累计时长、阅读日历、书籍分布、喜爱书籍、评分分布、格式分布 |
| | PDF工具栏 | 缩放/旋转/搜索/打印/首页/末页，浮动/固定样式，透明度调节 |
| | 底部工具栏 | 目录/上一页/下一页/设置/搜索 |
| | 全文搜索 | 搜索书籍内容，高亮显示结果，支持跳转 |
| **🖊️ 标注** | 颜色标注 | 7种颜色（红🔴橙🟠黄🟡绿🟢粉🩷蓝🔵紫🟣）|
| | 标注样式 | 高亮/下划线/边框/波浪线 |
| | 快速标注 | 工具栏选择颜色和样式，选中文本即可快速标注 |
| | PDF高级标注 | 墨迹标注、形状标注（矩形/圆形/三角形）、填充功能 |
| | 笔记系统 | 为标注添加详细笔记、实时编辑、添加标签 |
| | 书签功能 | 添加/移除书签、书签列表管理 |
| | 行级备注 | 为段落添加行级备注 |
| | 标注管理 | 按颜色/章节筛选、时间/日期/章节排序、删除标注 |
| | 快捷发送 | 配置快捷文档列表（最多5个），一键发送标注 |
| | 自动同步 | 标注自动同步到绑定的思源文档（添加/删除/修改） |
| | 撤销标注 | Ctrl+Z 撤销最近的标注 |
| | 复制设置 | 自定义链接格式，可用变量：书名/作者/章节/位置/链接/文本/笔记/截图 |
| | 精准定位 | 使用 CFI/页码精确定位，支持跳转到原文 |
| **🔊 朗读** | TTS模式 | Edge TTS（在线免费）、本地浏览器（离线） |
| | 多语音支持 | 数百种在线语音和本地语音，支持收藏常用语音 |
| | 智能播放 | 循环播放选中文本、从选中段落播放、从当前页播放、选中文本朗读 |
| | 精准高亮 | PDF精准高亮当前文本、EPUB自动滚动到当前段落 |
| | 播放控制 | 暂停/继续、快进/快退10秒、自动翻页、自动停止 |
| | 语音参数 | 语速/音量/音调调节 |
| | 播放选项 | 自动播放、高亮朗读文本、自动翻页 |
| **📚 书架** | 分组管理 | 文件夹分组、智能分组（按标签/格式/状态/评分自动筛选） |
| | 排序方式 | 最近阅读/添加时间/阅读进度/评分/时长/书名/作者/最近更新（全开放） |
| | 视图模式 | 网格/列表/紧凑（全开放） |
| | 多维筛选 | 状态/评分/格式/标签/更新状态（全开放） |
| | 书籍管理 | 编辑书籍信息（标题/作者/封面/评分/状态/标签）、移除书籍 |
| | 文档绑定 | 绑定思源文档，标注自动同步 |
| | 批量操作 | 批量转换EPUB样式、批量调整宽度 |
| | 界面设置 | 封面大小调整（80-160px）、工具栏透明度调节 |
| | 更新检查 | 检查在线书籍更新 |
| | 添加书籍 | 本地文件（EPUB/PDF/TXT）、HTTP(S)链接、绝对/相对路径 |
| **🔍 搜书** | 在线书源 | 多书源并发搜索，内置安娜的档案/Project Gutenberg/Standard Ebooks |
| | 自定义书源 | 支持 JSONPath/CSS/XPath/JavaScript/Regex |
| | 规则组合 | 支持 `&&`/`||`/`%%` 组合、`{$.path}` 嵌套、`@put/@get` 数据共享 |
| | 书源管理 | 导入/导出/启用/禁用/编辑/删除书源 |
| | 格式筛选 | 按格式筛选搜索结果 |
| | 快速添加 | 搜索结果一键加入书架 |
| | 章节搜索 | 搜索书籍章节内容 |
| **🎴 闪卡** | Anki导入 | 完整 .apkg 文件导入，保留卡组结构和卡片内容 |
| **🎴 闪卡** | 基础闪卡 | Anki导入、间隔重复、卡组管理、学习设置、数据统计 |
| | 思源闪卡同步 | 导入思源闪卡，实时双向同步（添加/删除/修改自动同步） |
| | FSRS算法 | 高级记忆算法，自动计算卡片稳定性和难度 |
| | 模板编辑 | 查看和编辑卡组模板（正面/背面/CSS）、实时预览 |
| | 高级搜索 | 支持卡组/标签/状态/属性多维筛选 |
| | 特殊卡片 | 图片遮挡、LaTeX公式渲染、填空支持 |
| **📖 词典** | 在线词典 | 7个词典源（剑桥/有道/海词/汉字/词语/汉典/必应） |
| | 离线词典 | 支持 StarDict 和 dictd 格式 |
| | 智能识别 | 自动选择最合适的词典 |
| | 词典管理 | 添加/删除离线词典 |
| | 加入卡组 | 将单词添加到卡组进行复习 |
| **🌐 翻译** | 翻译服务 | Azure/Google/Yandex/AI翻译(免费)/AI翻译(思源) |
| | 选中翻译 | 选中文本后直接翻译 |
| | 翻译面板 | 独立翻译面板显示结果 |
| **⚙️ 其他** | 授权系统 | 体验会员/月付会员/年付会员/永久会员 |
| | 快捷键 | 自定义快捷键（上一页/下一页/书签/PDF操作等） |
| | 数据管理 | SQLite统一数据管理、自动迁移旧版本数据 |
| | 移动端支持 | 支持 PDF 阅读（EPUB/TXT 暂不支持） |

## 权限设计原则

### 🆓 免费版
**定位**：基础阅读体验，满足日常阅读需求

**开放功能**：
- ✅ EPUB/PDF/TXT/在线小说 完整阅读
- ✅ 默认主题
- ✅ 所有阅读模式（单页/双页/连续滚动）
- ✅ 完整文本/布局/视觉设置
- ✅ 目录导航、脚注识别、全文搜索
- ✅ 完整PDF工具栏
- ✅ 简洁统计（本次/今日/累计时长）
- ✅ 完整标注功能（7色/4种样式/笔记/书签/行级备注/标注管理/自动同步/撤销）
- ✅ 基础书架（网格/列表/紧凑视图、8种排序、多维筛选）
- ✅ 2个在线词典（有道/必应）
- ✅ 完整闪卡功能（Anki导入、间隔重复、卡组管理、学习设置、数据统计）

**限制功能**：
- ❌ 主题配色（8种预设主题 + 自定义）
- ❌ 阅读统计详情（日历/分布图/喜爱书籍等）
- ❌ 快速标注
- ❌ 快捷发送
- ❌ TTS朗读
- ❌ 文件夹分组、智能分组
- ❌ 书籍编辑、批量操作、高级添加
- ❌ 在线书源搜索
- ❌ 高级词典（剑桥/海词/汉字/词语/汉典）
- ❌ 离线词典
- ❌ 翻译功能
- ❌ 文档assets同步
